Dual-Layer NLP Moderation Engine

Combining deterministic dictionary scanning with semantic AI to ensure a safe reporting environment.

[Production Run Metrics]
Total Qualitative Comments Processed: 116,737
1. Flagged by Dictionary rules: 7,012 (6.01%)
2. Unique Flags caught ONLY by AI: 664 (0.57%)

[Engineering Impact]
The 664 unique AI flags demonstrate the critical value of semantic screening. While dictionaries catch explicit profanity, the Claude 3 Haiku model successfully identified nuanced, passive-aggressive hostility and contextual abuse that hardcoded rules completely missed.

AWS Bedrock Batch Architecture

A highly robust pipeline designed to process massive datasets while bypassing API rate limits and JSON truncation errors.

[Pipeline Architecture Flow]

[1] Normalized Excel Dataset
 ↓
[2] Data Chunking & Aggregation (Max 3,500 - 4,000 chars to prevent token overflow)
 ↓
[3] Format to strict .jsonl payload & Upload to AWS S3
 ↓
[4] Trigger AWS Bedrock Asynchronous Batch Job (anthropic.claude-3-haiku)
 ↓
[5] Auto-Download, Parse JSON responses, and execute On-Demand Retries for any failures
 ↓
[6] Merge DataFrames for final Dashboard delivery
